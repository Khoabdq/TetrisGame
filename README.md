# Tetris Game

A classic, arcade-style Tetris game built from scratch. Drop, rotate, and clear lines to achieve the highest score possible before the grid fills up.

## 🚀 Features
- **Classic Gameplay:** Standard 10x20 grid with random, falling tetromino pieces.
- **Scoring System:** Points awarded for single, double, triple, and Tetris line clears.
- **Speed Progression:** Game speed increases automatically as the player scores higher.
- **Next Piece Preview:** A dedicated UI panel showing the upcoming block shape.
- **Controls:** Smooth keyboard inputs for movement, rotation, and hard drops.

## 🎮 How to Play
1. **Left / Right Arrows:** Move the falling tetromino horizontally.
2. **Up Arrow:** Rotate the piece 90 degrees clockwise.
3. **Down Arrow:** Soft drop (speed up descent).
4. **Spacebar:** Hard drop (instantly place the piece at the bottom).
5. **P Key:** Pause / Resume the game.
